# 异常处理器
#java #SSM 

---
## 异常常见位置以及诱因
- 框架内部抛出异常：因为使用不合规范
- 数据层抛出异常：因为内部服务器故障导致（例如：服务器访问超时）
- 业务层抛出异常：因业务逻辑书写错误导致（例如：遍历业务书写操作，导致索引异常等）
- 表现层抛出异常：因数据收集、校验等规则导致（例如：不匹配的数据类型件导致异常）
- 工具类抛出异常：因工具类书写的不够健壮导致（例如：必要时放的资源没有释放）

`所有异常都往上抛，到表现层进行处理`
