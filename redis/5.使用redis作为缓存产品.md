# 使用redis作为缓存
#redis #java #cache 

---
## Spring Cache使用方式
1. 导入Maven坐标
spring-boot-starter-data-redis、spring-boot-starter-cache
2. 配置application.yml
```yml
spring:
	cache:
		redis:
			time-to-live: 1800000 #设置有效期
```
3. 在启动类上加上注解@EnableCaching注解，开启缓存注解功能
4. 在Controller的方法上加入@Cacheable、@CacheEvict等注解，进行缓存操作
